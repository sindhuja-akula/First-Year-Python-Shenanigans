# 🐍 First-Year Python Shenanigans

Welcome to the digital museum of my very first Python projects! 

As a second-year engineering student shifting my focus toward Data Science and AI, I decided to dig up the code that started it all. This repository is a raw, unfiltered look at my early coding days—complete with questionable logic, wild loops, and a lot of passion.

## 📂 What's in the Archive?

Here is a quick guide to the masterpieces you will find in this repo:

* **`guess_number.py`**: Features an absolutely massive, hardcoded ASCII art intro that takes up half the file. 
* **`true_love.py`**: A flawless algorithm that calculates compatibility based purely on counting the letters in the words "true" and "love". Math at its finest.
* **`tic_tac_toe_better_version.py`**: A Tic-Tac-Toe game where I decided to import `numpy` for absolutely no reason other than making an array. 
* **`rock_pape_scissors.py`**: A classic game, featuring the legendary spelling of "sicssors". 
* **`pizza_order.py` & `pay_bills.py`**: Essential survival tools for a college student, built with raw dictionaries and lists.
* **`hungman_game.py`**: Includes some truly creative string manipulation and ASCII stick figures to keep you guessing.

## ⚠️ Disclaimer

This code is provided exactly as it was written in my first year. 
* Bugs are considered features.
* `while True` loops are abundant.
* Do not attempt to use this code in a production environment (especially the True Love calculator).

Every expert was once a beginner who couldn't figure out their indentation. We all have to start somewhere!
